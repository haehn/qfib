Parameters

outputFileNamePath of the output file (qfib)
outOfCoreBoolean : true when outOfCore, false otherwise
timingsWhen true, display the compression timings
displayErrorWhen true, compute the compression error and diplay it - only in in-core mode
displayDetailedErrorWhen true, compute the compression detailed error per fiber length and display it - only in in-core mode
verboseWhen true, display the maximum angle of the set
verifWhen true, verify that segments length are all the same length with a 10% margin - only in in-core mode Deactivate for full performance
faPath of an fa file to use for comparison - only in in-core mode If fa=="", no fa comparison is performed
zipTo automatically zip the output after compression for additional compression ratio
